# Amazon Reviews Sentiment Analysis Using NLP

## Project Overview

This project performs sentiment analysis on Amazon customer reviews to classify opinions as Positive, Negative, or Neutral. Natural Language Processing (NLP) techniques are used to understand customer sentiment and generate business insights.

## Objectives

- Classify customer reviews into sentiments.
- Apply NLP techniques for sentiment analysis.
- Analyze customer opinions and trends.
- Generate business insights from review data.
- Support decision-making through sentiment analysis.

## Tools and Technologies

- Python
- Pandas
- Matplotlib
- Seaborn
- TextBlob
- WordCloud
- Jupyter Notebook

## Dataset Information

- Rows: 2000
- Columns: 5

### Features:
- ReviewID
- Category
- Review
- Rating
- Sentiment

## Visualizations

- Sentiment Distribution
- Ratings Distribution
- Category-wise Sentiment
- Word Cloud
- Polarity Analysis

## Key Findings

- Most reviews are positive.
- Customer satisfaction is generally high.
- Negative reviews indicate areas for improvement.
- Sentiment analysis helps understand customer opinions.

## Business Insights

- Improve product quality.
- Enhance customer experience.
- Support marketing strategies.
- Understand customer needs.

## Conclusion

The project successfully applied NLP techniques to analyze customer reviews and generate valuable insights for business decision-making.
